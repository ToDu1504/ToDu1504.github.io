Làm quen với ngôn ngữ lập trình Java

October 20, 2025 - One minute read - 74 words

Java là một ngôn ngữ lập trình hướng đối tượng phổ biến được phát triển bởi Sun Microsystems. Điểm mạnh của Java là tính độc lập nền tảng — chương trình viết một lần có thể chạy trên nhiều hệ điều hành khác nhau nhờ máy ảo JVM.

Một chương trình Java cơ bản có cấu trúc như sau:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào Java!");
    }
}
