Những tính năng ES6 quan trọng cần biết

October 20, 2025 - One minute read - 42 words

ES6 (ECMAScript 2015) mang lại nhiều cải tiến giúp code ngắn gọn, dễ đọc hơn:

  • let, const thay var
  • Arrow function
  • Template string
  • Destructuring
  • Modules

Ví dụ:

const name = "An";
const sayHello = () => {
    console.log(`Xin chào ${name}`);
};
sayHello();
