Tổng hợp một số thuật toán cơ bản thường gặp

Thuật toán là nền tảng của lập trình. Một số thuật toán cơ bản bạn nên biết:

  1. Tìm min/max trong mảng
  2. Sắp xếp (Bubble, Selection, Insertion)
  3. Tìm kiếm tuyến tính / nhị phân
  4. Đảo chuỗi, đếm số lần xuất hiện…

Ví dụ (JavaScript):

const arr = [5, 3, 9, 1];
let max = arr[0];
for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
}
console.log("Max =", max);

Tìm hiểu Async/Await trong JavaScript

Async/Await giúp viết code bất đồng bộ (asynchronous) dễ đọc như đồng bộ.

Ví dụ:

function wait(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
}

async function run() {
    console.log("Bắt đầu");
    await wait(2000);
    console.log("Hoàn thành sau 2 giây");
}

run();

Thao tác DOM trong JavaScript

DOM (Document Object Model) cho phép JS tương tác trực tiếp với nội dung HTML.

Ví dụ:

<p id="greeting">Xin chào!</p>
<button onclick="changeText()">Đổi</button>

<script>
function changeText() {
    const p = document.getElementById("greeting");
    p.textContent = "Chào bạn lập trình viên!";

Các kiểu dữ liệu trong JavaScript

JavaScript có hai nhóm chính: primitive và object.

  • Primitive: number, string, boolean, null, undefined, bigint, symbol.
  • Object: bao gồm array, function, object…

Ví dụ:

let age = 20; // number
let name = "An"; // string
let isStudent = true; // boolean
