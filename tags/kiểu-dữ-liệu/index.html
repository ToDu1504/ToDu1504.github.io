Các kiểu dữ liệu trong JavaScript

JavaScript có hai nhóm chính: primitive và object.

  • Primitive: number, string, boolean, null, undefined, bigint, symbol.
  • Object: bao gồm array, function, object…

Ví dụ:

let age = 20; // number
let name = "An"; // string
let isStudent = true; // boolean
let person = { name: "An", age: 20 }; // object

Các kiểu dữ liệu cơ bản trong Java

Java có hai nhóm kiểu dữ liệu: kiểu nguyên thủy và kiểu tham chiếu.

  • Kiểu nguyên thủy: gồm 8 kiểu như int, double, boolean, char… Dùng để lưu trữ giá trị đơn giản.
  • Kiểu tham chiếu: là các đối tượng như String, Array, Class…

Ví dụ:

int age = 20;
double pi = 3.14;
boolean isStudent = true;
char grade = 'A';
String name = "An Nguyen";
